# Curso de introdução à anotação de variantes.

Este repositório contém as instruções para a parte prática do curso de introdução à anotação de variantes ministrado para a 1a Semana de Bioinformática do RSG-Brazil.

## Proposta

O objetivo deste curso é trazer noções básicas de anotação de variantes na prática utilizando ferramentas e bancos de dados públicos. Para isso, utilizaremos um dataset limitado ao cromossomo 20 para fins didáticos e implementaremos todas as etapas necessárias para montar um pipeline de anotação.
